Test1

Test1

Test1
Request Type HTTP GET
URL Params

Field Type-FieldName

  • Integer-MerchGroupID [Required]
    • Description: Merchandiser GroupID for Group Name like Lewisville, MidCities, Minneapolis etc. (SDM Table Name - [Setup].[MerchGroup])
  • String-GSN [Required]
    • Description: Merchandiser GSN
  • Date-StartDispatchDate [Optional]
    • Description:- Start schedule date, if not provided the current date will be used. The service will provide schedule for the next 4 weeks(configurable from the server side) starting from this date.
  • Date-LastScheduleDate [Optional]
    • Description: Merchendiser Last scheduled date. This is the last day when MyDay has schedule for the GSN in the context. From StartDispatchDate to LastScheduleDate, MyDay has the schedule as current as the BatchID indicated with the next parameter. So the service will only need to provde differential updates for this range. For any day passed LastScheduleDate, the service will provide full load.
  • Integer-BatchID [Optional]
    • Description: Used to identify the downloaded scheduled data in Merchendiser device, as explained above.
