Method Name UploadDeliveryTime
URL

/api/Merch/UploadDeliveryTime

Test URL

Test1
Request Type HTTP POST
JSON Params

Below fields needed in JSON format

Field Type-FieldName

  • DateTime-DeliveryDate
    • Description: Driver delivery date
  • string-RouteNumber
    • Description: Route Number as defined in RouteManager and Roadnet
  • string-DriverID
    • Description: DriverID as defined in Roadnet
  • string-GSN
    • Description: Driver GSN
  • Array-DeliveryStops Description: The list of stops that either actual delivery time is captured, and/or new estimated arrival time is available
    • Integer-StopSequence
      • Description: StopSequence as defined in delivery plan
    • Integer-StopID
      • Description: StopID as defined in delivery plan
    • DateTime-ActualArrivalTime
    • string-ActualArrivalTimeZone
    • DateTime-ActualDepartureTime
    • string-ActualDepartureTimeZone
    • DateTime-EstimatedArrivalTime
    • string-EstimatedArrivalTimeZone
